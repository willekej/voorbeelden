---------------------------------------------------------------------
Welke Markup in HTML?
- doctype     <!DOCTYPE html>
- elements    <p> ... </p>, <br>, <br /> 
- attributes  <p class="intro"> ... </p>
- entities    &lt;  
            als a<b   als a&lt;b
- commentaar  <!-- ... -->
---------------------------------------------------------------------
2 soorten elementen (voorlopig)
- block elements: h1, p, div 
        (standaard gedrag: zo breed mogelijk, onder elkaar gestapeld)
- inline elements: strong, em, a, span
        (standaard gedrag: zo breed als de inhoud, deel van de tekst))
---------------------------------------------------------------------
nut div, span?: via CSS vormgeven
<div style="border:1px solid black">...</div>
<span style="text-decoration:underline">...</span>
---------------------------------------------------------------------

    position:static;
        het element komt op zijn normale positie (default waarde v elk element)
        left, right, top, bottom wordt genegeerd
    position:relative;
        originele ruimte van het element blijft behouden
        positioneerbaar via left, right, top, bottom
        positioneerbaar tov originele ruimte van het element
    position:absolute;
        originele ruimte van het element wordt vrijgegeven (taken out of document flow)
        positioneerbaar via left, right, top, bottom
        positioneerbaar
            tov scherm indien het element geen block ancestor heeft die 
              niet static gepositioneerd is
            tov eerste block ancestor die niet static gepositioneerd is anders
    position:fixed;
        originele ruimte van het element wordt vrijgegeven (taken out of document flow)
        positioneerbaar via left, right, top, bottom
        scrolt mee met scherm
    position: sticky (tussen relative en fixed)
